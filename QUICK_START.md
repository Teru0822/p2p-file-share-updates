# P2P File Share v2.2.0 - クイックスタートガイド

## 🚀 5分でスタート

### 1. インストール（初回のみ）

```bash
# プロジェクトフォルダに移動
cd p2p-file-share

# 依存関係をインストール
npm install

# ファイアウォール設定（管理者権限で実行）
firewall-setup.bat
```

### 2. アプリ起動

```bash
npm start
```

起動すると自動的に：
- ✅ 開発者ツールが開く
- ✅ ネットワークをスキャン開始
- ✅ 同じLAN内のデバイスを検出

### 3. 使ってみる

#### ファイル送信
1. デバイス一覧で送信先を選択
2. 「📤 送信」ボタンをクリック
3. ファイルを選択
4. 自動的に転送開始

#### テキスト送信
1. デバイス一覧で「💬 テキスト」ボタンをクリック
2. メッセージを入力
3. 「送信」ボタンをクリック
4. 相手に即座に届く

#### ドラッグ&ドロップ
- ファイルをデバイス上にドロップするだけで送信

## 🔧 よくある問題

### Q: デバイスが見つからない

**A: ファイアウォール設定を確認**
```bash
firewall-setup.bat
```
管理者権限で実行してください。

### Q: 開発者ツールが出ない

**A: v2.2.0では自動で開きます**
- `main.js`に`openDevTools()`が含まれているか確認
- 起動時のConsoleを確認

### Q: テキストが送れない

**A: Consoleでエラー確認**
1. F12またはCtrl+Shift+I
2. Consoleタブを確認
3. エラーメッセージを読む

### Q: アップデート通知が繰り返し出る

**A: localStorageをクリア**
```javascript
// Consoleで実行
localStorage.clear();
```
その後、アプリを再起動

## 🎯 次のステップ

```bash

```
`dist/`フォルダに実行ファイルが作成されます。

### GitHub Pagesで配布
1. index.htmlをGitHubリポジトリにpush

3. アプリが自動的に更新を検知

- `index.html`のVERSION_INFOを編集
